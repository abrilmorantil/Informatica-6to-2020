¿Cómo hice esta página?

Esta página fue creada utilizando HTML para estructurar el contenido y CSS para darle estilo. HTML permitió organizar la información y definir la disposición de las secciones, mientras que CSS se utilizó para aplicar estilos visuales, como la elección de colores, tamaños de texto y el diseño general de la página. El uso de CSS también permitió hacer que el contenido fuera visualmente atractivo y fácil de leer, mejorando la experiencia del usuario.
